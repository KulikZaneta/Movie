# Movie 
Movie application

## General info
Application allows to read details about movies contained in database.
Project under reconstruction.

## Technologies
* Java 8
* Spring Boot 2.1.6
* Hibernate 5.3.10
* Database: H2, Elasticsearch 6.2.2
* Swagger 2.9.2

## Build
1. Clone and unpack application

2. Run elasticeasrch (version 6.2.2)

3. Clone and unpack Angular application -> https://github.com/KulikZaneta/MovieAngular
